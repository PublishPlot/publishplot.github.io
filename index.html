Publication-Quality Plots, Effortlessly

PublishPlot transforms any text-based table of data into stunning, publication-ready plots. Designed by a professor and journal editor frustrated with subpar visuals like those from Excel, it’s the essential tool for students, teachers, professors, scientists, and engineers. Simply copy-paste or drag-and-drop your data—high-quality results appear instantly.

Key Advantages

Relative Sizing for Perfect Scalability

All elements size relative to the plot diagonal, mimicking traditional illustrator techniques. Symbols at 1-2% of diagonal, fonts scaled virtually for a 10-inch plot—everything adapts flawlessly when resizing for publications.

Seamless Customization

Tweak preferences for initial plots or fine-tune every detail post-creation. Add error bars, text labels, arrows. Export as PDF via drag-and-drop to any app or desktop.

Scriptable and Extensible

Control via AppleScript or Python. Built-in data transformations and fits, plus custom scripting for complex functions. Integrates plotting into your custom software workflows.

Post-Processing Focus

Basic in-app data editing keeps it lightweight. Ideal for polishing data from spreadsheets, text files, or custom tools—edit at source, plot here.

Fast, Reliable, Proven

Version 3.3.1 ensures full color compatibility on macOS Tahoe and beyond. Get great plots fast, without compromise.

How It Works

1. Import Data

Copy-paste from spreadsheets or drag-and-drop text files into PublishPlot. Instant plot generation.

2. Customize

Adjust via preferences or directly: error bars, annotations, relative sizes—all intuitive and precise.

3. Export
